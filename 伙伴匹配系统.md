字段：

|   字段名    |   类型   |            说明            |
| :---------: | :------: | :------------------------: |
|     id      |   int    |            主键            |
|  tag_name   | varchar  |           标签名           |
|   user_id   |   int    |      上传标签的用户id      |
|  parent_id  |   int    |          父标签id          |
|  is_parent  | tinyint  | 是否为父标签(0-不是，1-是) |
| create_time | datetime |          创建时间          |
| update_time | datetime |          更新时间          |
|  is_delete  | tinyint  |      是否删除(1-删除)      |

为tag_name增加唯一索引
为user_id增加普通索引
